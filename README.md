# Stock Price Tracker Automation using Python

## Project Overview
The **Stock Price Tracker** is a Python-based automation project that monitors real-time stock prices and sends an **email alert** when a target price is reached.  
It uses the **Yahoo Finance API** via the `yfinance` library and runs continuously using a **scheduler**.

This project demonstrates:
- Real-time data fetching
- Automation using schedulers
- Alerting systems via email
- Practical use of Python for financial monitoring

---

## Objectives
- Fetch real-time stock prices automatically  
- Monitor stock prices at regular intervals  
- Trigger an email alert when the target price is met  
- Run continuously with minimal manual intervention  

---

## Technologies Used
- **Python**
- **yfinance** – Fetch real-time stock market data
- **schedule** – Task scheduling
- **smtplib / email** – Email notifications
- **GitHub** – Version control and project hosting
